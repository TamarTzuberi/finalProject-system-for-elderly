# Eldery - Front

This repository contains the frontend for a web application designed for researchers studying the elderly at Ben Gurion University of the Negev. The frontend connects to the server and displays data from the MongoDB database, as well as data from the Google Fit API and Xiaomi Band 6.

### Getting Started
Clone the repository to your local machine
`git clone https://github.com/YOUR-USERNAME/finalProject-system-for-elderly-frontend.git`

Install the necessary dependencies
`npm install`

Start the development server
`npm start`

Connect to the server by updating the connection string in the config file.

### Features
Display data from the MongoDB database, Google Fit API, and Xiaomi Band 6 in graph format.
Compare subjective data with objective data.
Extract data to CSV from different dates.

### Usage
The frontend includes the following pages:

Data visualization page: Displays data in graph format.

Comparison page: Allows for comparison of subjective and objective data.

Data extraction page: Allows for extraction of data to CSV from different dates.

### Note
The frontend of the application is designed to work with the server. Therefore, the server must be running and configured correctly for the frontend to work properly.
